# Event Service

Микросервис **event-service** управляет событиями пользователей в проекте **ExploreWithMe**.  
Отвечает за создание, обновление, публикацию и получение информации о событиях.

---

## Основные функции

- Создание и редактирование событий пользователями  
- Публикация событий администраторами  
- Получение списка событий с фильтрацией и пагинацией  
- Подсчёт подтверждённых заявок и просмотров  
- Проверка принадлежности события пользователю  

---

## Взаимодействие с другими сервисами

| Сервис | Назначение | Протокол |
|--------|-------------|----------|
| `user-service` | получение данных о пользователях | Feign |
| `request-service` | обработка заявок на участие | Feign |
| `comment-service` | управление комментариями | Feign |
| `gateway` | приём внешних HTTP-запросов | REST |

Общие DTO и клиенты определены в модуле `interaction-api`.

---

## Технологии

- Java 21  
- Spring Boot 3.3  
- Spring Data JPA  
- Spring Cloud (OpenFeign, Discovery, Config)  
- PostgreSQL  
- Lombok  
- Maven  

---

## Конфигурация

`event-service` регистрируется в Eureka и использует базу данных `eventservice`.  
Конфигурация загружается из `config-server`.  
DDL формируется автоматически на основе JPA-моделей.

Обеспечивает хранение, создание, обновление и удаление пользовательских данных.

# Interaction API

Модуль **interaction-api** содержит общие классы данных (DTO) и интерфейсы Feign-клиентов,  
используемые всеми микросервисами проекта **ExploreWithMe**.

---

## Основное содержимое

- DTO для пользователей, событий, заявок и комментариев  
- Классы запросов и ответов REST API  
- Интерфейсы Feign-клиентов для межсервисного взаимодействия  

---

## Назначение

Обеспечивает единый контракт обмена данными между микросервисами.  
Используется как зависимость во всех сервисах (`event-service`, `user-service`, `comment-service`, `request-service`, `gateway`).

---

## Технологии

- Java 21  
- Spring Boot 3.3  
- Spring Cloud OpenFeign  
- Lombok  
- Maven  

---

## Конфигурация

Не является самостоятельным приложением.  
Собирается как библиотека и подключается к другим сервисам через Maven-зависимость.
